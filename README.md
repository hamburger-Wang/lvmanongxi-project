# 农业智能分析系统
轻量多尺度遥感特征融合的作物智能识别系统

## 项目概述

### 1.1 项目背景
随着精准农业的发展，卫星/无人机遥感数据的规模化应用成为农业生产决策的核心支撑。本项目面向农业生产场景，开发集**数据导入、作物分类、长势预测、种植建议**于一体的可视化桌面应用，解决传统农业分析流程繁琐、模型调用门槛高的问题，为农业从业者提供高效、易用的智能分析工具。

### 1.2 核心目标
- **降低使用门槛**：通过可视化界面实现无代码化操作，让非技术人员也能轻松使用AI模型
- **多源数据支持**：支持卫星/无人机/融合数据的批量导入与解析
- **智能分析功能**：集成作物分类与长势预测模型，输出可落地的种植建议
- **流畅用户体验**：通过多线程实现模型异步运行，避免界面卡顿

### 1.3 适用场景
- **农业科研机构**：作物生长态势分析、模型效果验证
- **农业生产企业**：规模化农田的产量预测、精准管理决策
- **大创/毕设项目**：农业AI+可视化方向的完整工程实践
- **学生学习**：了解如何将AI模型与桌面应用结合的实践案例

## 快速开始

### 2.1 环境搭建
1. **安装Anaconda**：从[Anaconda官网](https://www.anaconda.com/)下载并安装
2. **创建虚拟环境**：
   ```bash
   conda create -n agri_project python=3.9
   conda activate agri_project
   ```
3. **安装依赖**：
   ```bash
   pip install PySide6 pandas numpy h5py tifffile
   ```

### 2.2 运行项目
1. **克隆项目**：
   ```bash
   git clone <项目地址>
   cd lvmanongxi-project
   ```
2. **运行应用**：
   ```bash
   python main.py
   ```
3. **使用流程**：
   - 启动应用后，可选择跳过登录直接进入主界面
   - 在侧边栏选择功能模块：地图分析、作物预测、种植建议
   - 导入数据文件，配置模型参数，运行分析
   - 查看分析结果和AI建议

## 项目结构

### 3.1 文件说明

| 文件路径 | 作用 | 功能描述 |
|---------|------|----------|
| **main.py** | 程序入口 | 启动应用、配置全局字体，创建并显示主窗口 |
| **ui_main.py** | 主窗口核心 | 整合登录/主界面，实现侧边栏模块切换、菜单栏功能 |
| **ui_login.py** | 登录窗口 | 支持跳过登录，处理登录状态切换 |
| **config.py** | 全局配置 | 管理字体、颜色、文件路径、支持的格式等常量 |
| **crop_model.py** | 核心算法 | 作物分类/预测核心算法脚本，接收参数执行模型训练/推理 |
| **ui_modules/map_widget.py** | 地图分析模块 | 地图分析界面，预留卫星/无人机地图可视化接口 |
| **ui_modules/crop_widget.py** | 作物预测模块 | 作物预测核心界面，实现数据导入、参数配置、模型调用、结果展示 |
| **ui_modules/advice_widget.py** | 种植建议模块 | 种植建议界面，预留AI建议生成接口 |
| **core/data_importer.py** | 数据导入工具 | 解析多格式文件、获取文件详情、校验数据有效性 |
| **core/crop_worker.py** | 模型异步线程 | 调用crop_model.py，反馈日志/进度/结果，避免界面卡顿 |
| **core/loading_thread.py** | 通用加载线程 | 处理数据加载进度，复用在各类需要进度展示的场景 |

### 3.2 核心功能模块

1. **地图分析模块**
   - 支持卫星/无人机数据可视化
   - 区域选择和信息显示
   - 地图控制（放大、缩小、切换图层）

2. **作物预测模块**
   - 多格式数据导入（HDF5、TIFF、IMG等）
   - 模型参数配置（损失函数、训练轮数等）
   - 模型训练和预测
   - 结果展示和日志输出

3. **种植建议模块**
   - 常见问题选择
   - 自定义问题输入
   - AI种植建议生成

## 使用指南

### 6.1 数据导入
1. 在作物预测模块中，点击"选择文件导入"按钮
2. 选择要导入的数据文件（支持HDF5、TIFF、IMG等格式）
3. 查看文件详情，确认数据正确导入

### 6.2 模型配置与运行
1. 选择运行模式（训练/预测）
2. 选择损失函数（Cross-entropy/IOU/F1）
3. 设置训练轮数和模型路径
4. 点击"运行分类模型"按钮开始执行
5. 查看运行日志和进度条

### 6.3 结果查看
1. 模型运行完成后，在结果区域查看预测结果
2. 查看生成的结果图（如果有）
3. 可在种植建议模块中获取基于预测结果的种植建议

## 常见问题

### 7.1 运行时错误
- **问题**：无法启动应用
  **解决方案**：检查Python版本是否为3.9+，确保所有依赖已正确安装

- **问题**：模型运行失败
  **解决方案**：检查数据文件格式是否正确，确保模型路径存在

- **问题**：界面卡顿
  **解决方案**：关闭其他占用系统资源的程序，确保计算机性能足够

### 7.2 数据导入问题
- **问题**：无法导入数据文件
  **解决方案**：检查文件格式是否支持，确保文件路径正确

- **问题**：文件详情显示错误
  **解决方案**：尝试使用其他格式的文件，或检查文件是否损坏

## 扩展与定制

### 8.1 添加新模型
1. 在`crop_model.py`中实现新模型
2. 在`ui_modules/crop_widget.py`中添加模型选择选项
3. 更新配置文件中的模型路径

### 8.2 自定义界面
1. 修改`config.py`中的颜色和字体配置
2. 在各UI模块中调整布局和样式
3. 添加新的功能按钮和交互元素

### 8.3 接入真实AI模型
1. 准备训练好的模型文件
2. 修改`crop_model.py`中的模型加载和推理逻辑
3. 配置模型路径和参数
